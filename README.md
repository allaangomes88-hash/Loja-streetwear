<h1>Drip</h1>
<p>Loja streetwear fictícia criado pra praticar JS puro, manipulação do DOM, eventos, arrays, funções, e simulação de uma API com dados vindos de um backend fake.</p>

<h2>Como funciona</h2>
<p>O projeto simula uma API real: o arquivo db.json funciona como banco de dados com os produtos, e o api.js expõe funções para buscar esses dados, imitando o que seria uma requisição a um servidor.</p>
<p>O script.js consome essa "API" e renderiza os produtos dinamicamente no DOM, sem nenhum dado fixo no HTML.</p>
<h2>Desenvolvimento</h2>
<p>HTML e CSS foi feito pelo claude.ai</p>
